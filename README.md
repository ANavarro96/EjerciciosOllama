# IntroOllama

Proyecto de ejercicios basicos para practicar prompting y uso de Ollama desde Python.

## Requisitos

- Python 3.10+ (recomendado)
- Servidor de Ollama en local (por defecto: `http://localhost:11434`)
- Modelo disponible en Ollama (por ejemplo: `llama3.2:3b`)

## Instalacion

```powershell
conda init
conda activate
pip install -r requirements.txt
```

## Ejecucion

Ejemplo basico:

```powershell
python ejemploOllama.py
```

Tambien puedes ejecutar los ejercicios:

```powershell
python ejercicio1.py
python ejercicio2.py
python ejercicio3.py
python ejercicio4.py
python ejercicio5.py
python ejercicio6.py
```

## Dependencias

El proyecto usa estas librerias de Python (ver `requirements.txt`):

- `ollama`
- `jsonschema`
- `pydantic`
